## 📌 Project Objective
The goal of this project is to perform Exploratory Data Analysis (EDA) and answer key business questions using SQL to uncover patterns, trends, and actionable insights from the dataset.

This project focuses on:
- Developing strong SQL querying skills
- Understanding data distributions
- Creating KPI-driven dashboard mockups
- Presenting insights using visualizations

---

## 🗂️ Dataset Description
The dataset contains transactional sales data with the following key fields:

- Order Date  
- Sales  
- Profit  
- Quantity  
- Product Name  
- Category & Sub-Category  
- Customer Name  
- Region / State / Country  
- Discount  
- Ship Mode

## 📈 Project Workflow

### 1️⃣ Descriptive Statistics & Univariate Analysis
- Total Sales & Total Profit
- Average Order Value
- Sales distribution
- Category-wise order count
- Profit distribution

### 2️⃣ SQL for Business Questions
Key business questions solved:

- Top 5 products by revenue
- Monthly sales trend
- Most profitable category
- Top 10 customers by sales
- Region-wise performance
- Discount impact on profit

### 3️⃣ Multivariate Analysis
- Sales vs Profit relationship
- Discount vs Profit impact
- Quantity vs Sales trend

### 4️⃣ KPI Development
Proposed business KPIs:

- Total Sales
- Total Profit
- Total Quantity
- Monthly Growth
- Top Products
- Region-wise Sales
