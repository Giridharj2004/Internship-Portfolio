# 📊 Deep-Dive Analysis & Interactive Dashboarding

## 🚀 Internship Task 3 – ApexPlanet Software Pvt. Ltd.

This repository contains my work for **Task 3: Deep-Dive Analysis & Dashboarding**, completed as part of my **60-Day Data Analytics Internship**.

This phase focuses on transforming analyzed data into **business-driven insights and interactive dashboards**.

---

## 📌 Objective

To perform **advanced data analysis** and build **interactive dashboards** that help stakeholders make informed business decisions.

---

## 📂 Dataset Used

- Global Store Sales Dataset  
- Contains data on:
  - Sales
  - Profit
  - Discounts
  - Customer Segments
  - Regions
  - Categories & Sub-Categories
  - Order Dates

---

## 🧠 Key Analysis Performed

### 🔹 1. Segmentation Analysis
- Region-wise performance
- Customer segment contribution
- Category & sub-category breakdown

### 🔹 2. Profitability Analysis
- Identified profit-making vs loss-making products
- Analyzed impact of discounts on profit

### 🔹 3. Sales Trend Analysis
- Monthly & yearly trends
- Seasonal patterns

### 🔹 4. Business Insights Extraction
- High-performing regions and categories
- Areas needing improvement

---

## 📊 Dashboard Features

- 📈 Sales & Profit Overview  
- 🌍 Regional Performance Analysis  
- 🛍 Category & Sub-Category Insights  
- 👥 Customer Segment Analysis  
- ⏳ Time-Based Trends  

---

## 🛠 Tools & Technologies Used

- Python (Pandas, Matplotlib, Seaborn)
- SQL (for business queries)
- Power BI (Interactive Dashboard)
- Jupyter Notebook (VS Code)

## 📈 Key Insights

- Certain regions contribute significantly to overall sales but have low profitability  
- Discounts heavily impact profit margins  
- Specific categories drive the majority of revenue  
- Seasonal trends affect sales performance
